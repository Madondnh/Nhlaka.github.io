# NIPHO Portfolio

This repository is a static developer portfolio website based on the HTML5 UP "Editorial" template. It has been customized to showcase the skills, projects, and contact information of a freelancing software engineer named NIPHO.

## Features

- Responsive single-page layout with dynamic fragment loading
- Modular HTML fragments (`header.html`, `main_content.html`) assembled via JavaScript
- Sidebar navigation with education and contact sections
- Placeholder project images and descriptions
- Initialization logic in `assets/js/main.js` with safe checks

## Getting Started

1. Open the workspace in VS Code.
2. Install the **Live Server** extension (or use Python/Node static server).
3. Right-click `index.html` and select **Open with Live Server**.
4. Modify `header.html` or `main_content.html` to update content.
